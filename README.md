# Seaborn Data Visualization

## Aim

To learn and implement different data visualization techniques using the Seaborn library in Python.

## Description

This project demonstrates various types of plots provided by Seaborn. The visualizations are created using sample datasets and the Iris dataset to understand how graphical representations help in analyzing data.

## Plots Covered

* Line Plot
* Scatter Plot
* Box Plot
* Violin Plot
* Swarm Plot
* Bar Plot
* Point Plot
* Count Plot
* KDE Plot

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Scikit-Learn

## Requirements

```bash
pip install pandas matplotlib seaborn scikit-learn
```

## How to Run

1. Install the required libraries.
2. Open the notebook in Jupyter Notebook.
3. Run the code cells one by one or use Run All.
4. View the generated plots.

## Outcome

Successfully created and customized different Seaborn visualizations and learned how they can be used for data analysis.
